# 📝 AI Text Summarizer

A Streamlit web app that summarizes any text using AI. Supports pasting text, scraping URLs, and uploading PDF or `.txt` files.

## Features
- **Three input methods**: paste text, enter a URL, or upload a file (PDF / .txt)
- **Three output formats**: structured (title + summary + bullets), paragraph, or bullet points only
- Powered by LLaMA 3 via Groq API
